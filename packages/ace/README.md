ACE Editor
==========

These instructions cover how to include the Ajax.org Code Editor (ACE).

Homepage: [https://github.com/cadorn/ace](https://github.com/cadorn/ace)


Install
=======

package.json
------------

    {
        "mappings": {
            "ace": {
                "catalog": "http://registry.pinf.org/cadorn.org/github/catalog.json",
                "name": "ace",
                "revision": "master"
            }
        }
    }
